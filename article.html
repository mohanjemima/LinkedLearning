1. What is HTML?

What will be covered in this lesson?

This lesson focuses on what HTML is. It will give you the required knowledge to start your HTML journey and enable you to build a simple webpage!

What does HTML stand for?

  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of a Web Page
  • HTML consistent of a series of elements
  • HTML elements tell the browser how to display content

Here are what some common HTML elements look like:

  • <p>
  • <h1>
  • <a>
  • <html>
